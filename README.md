# figmaimmofit
In diesem Repo ist das Mockup zu der App Immofit welche im Rahmen des Moduls Projektmanagement (Fallstudie) für die Präsentation benötigt wurde.

  This is a code bundle for ImmoFit. The original project is available at https://www.figma.com/design/Yqqu5IV1HFetyYvJpvKqVo/ImmoFit.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
